# Insurance-Cost-Analysis
Identify key factorsinfluencing insurance premium costs and support data-driven pricing strategies. 
This project focused on analyzing insurance data to identify the key factors that influence premium costs and support data-driven pricing strategies.
I used Python libraries such as Pandas and NumPy for data cleaning and manipulation, and Matplotlib for visualizing trends. I began by cleaning and transforming the dataset, ensuring it was structured and ready for analysis. I then explored relationships between variables such as age, BMI, smoking habits, and insurance charges to understand how these factors impact premium costs.
Through exploratory data analysis, I identified strong correlations—particularly showing that smoking status and higher BMI significantly increase insurance costs. To make the insights more accessible, I created interactive dashboards in Tableau, which helped visualize trends, risk categories, and cost patterns clearly.
